# Injury Prediction Using AI

## Project Overview

This project uses Machine Learning techniques to predict the likelihood of injuries based on historical data. The model is trained using Python and implemented through Jupyter Notebook.

## Files

* injury prediction.ipynb - Model development and analysis
* injury_model.py - Python implementation of the model
* injury_model.pkl - Trained machine learning model

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Jupyter Notebook

## Objective

To develop an AI-powered injury prediction system that helps identify potential injury risks and supports preventive decision-making.
